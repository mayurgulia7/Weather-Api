# Weather-Api
A responsive weather application built with HTML, CSS, and JavaScript that fetches real-time weather data using a Weather API. Users can search any city to view current temperature, humidity, wind speed, and weather conditions through a clean and intuitive interface.
🌦️ Weather App

A responsive weather application built with HTML, CSS, and JavaScript that provides real-time weather information for any city using a Weather API. Users can search for a location and instantly view current weather conditions through a clean and intuitive interface.

✨ Features
🔍 Search weather by city name
🌡️ Displays current temperature
☁️ Shows weather condition and description
💧 Displays humidity levels
💨 Shows wind speed
📱 Responsive design for desktop and mobile
⚡ Fast and user-friendly interface
🛠️ Technologies Used
HTML5
CSS3
JavaScript (ES6)
Weather API (e.g., OpenWeatherMap API)
🚀 How to Run
Clone this repository.
Obtain an API key from your preferred weather service (such as OpenWeatherMap).
Add your API key to the JavaScript file.
Open index.html in your browser.
Enter a city name to view its current weather.
